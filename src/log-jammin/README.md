# LogJammin

A Jackie Treehorn production, care of the Big Lebowski.

## Description

A logging service I can share amongst microservices.

## Installation

Doesn't work on it's own, needs to be included into one of your other Nest.js APIs.

## Test

```bash
# run tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
